# Frontend — Campus Resource Sharing Platform  
*Vue.js + Element Plus + Bilingual UI*

This is the **frontend** of the Campus Resource Sharing System.

---

## Features
- Vue.js (SPA)
- Element Plus UI Components  
- Vue Router  
- Vue I18n (EN / 中文)  
- Axios for backend API  
- Responsive layout  
- User UI for services  
- Resource listing & detail pages  
- Admin interface for resource audit  

---

## Tech Stack
- Vue 2 
- Vue Router  
- Vue I18n  
- Element Plus  
- Axios  
- Node.js + NPM

## How to Run Locally

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
